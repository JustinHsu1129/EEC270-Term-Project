# Vim Cheat Sheet

| Category | Shortcut | Description |
| --- | --- | --- |
| **Navigation** | `h` `j` `k` `l` | Move cursor Left, Down, Up, Right |
|  | `w` / `b` | Jump to next / previous word |
|  | `0` / `$` | Move to start / end of line |
|  | `gg` / `G` | Move to start / end of file |
|  | `{` / `}` | Jump to previous / next paragraph |
| **Modes** | `i` | **Insert mode** (start typing) |
|  | `Esc` | **Normal mode** (navigation/commands) |
|  | `v` | **Visual mode** (highlighting text) |
|  | `V` | **Visual Line mode** (highlighting lines) |
|  | `:` | **Command mode** |
| **Editing** | `x` | Delete character at cursor |
|  | `dw` | Delete word |
|  | `dd` | Delete (cut) current line |
|  | `yy` | Yank (copy) current line |
|  | `p` | Paste after cursor |
|  | `u` | Undo last action |
|  | `Ctrl + r` | Redo last action |
| **Search** | `/pattern` | Search for "pattern" |
|  | `n` / `N` | Repeat search forward / backward |
|  | `:%s/a/b/g` | Replace all `a` with `b` in file |
| **Files** | `:w` | Save (write) |
|  | `:q!` | Quit without saving |
|  | `:wq` | Save and quit |
